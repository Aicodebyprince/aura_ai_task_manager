# **App Name**: Greeting Display

## Core Features:

- Text Input: Allow the user to input custom text to display.
- AI-Powered Text Suggestions: Suggest greetings based on user input as a tool.
- Display Greeting: Display the 'Hello, World!' or custom text in a prominent area.

## Style Guidelines:

- Primary color: Light grayish-blue (#A7C4BC) for a calm and friendly feel, inspired by approachability.
- Background color: Very light grayish-blue (#F0F4F3), a brighter variation of the primary to ensure high contrast.
- Accent color: Pale violet-red (#D0A2B7) to provide visual interest.
- Font: 'PT Sans', a modern humanist sans-serif suitable for both headlines and body text.
- Simple, centered layout to emphasize the text.
- Subtle fade-in animation on text display.
